# Levantamento de Requisitos

## Requisitos Funcionais

| ID | Requisito | Prioridade |
|---|---|---|
| RF01 | O sistema deve permitir cadastro de usuários. | Alta |
| RF02 | O sistema deve permitir login de usuários autorizados. | Alta |
| RF03 | O sistema deve permitir cadastrar áreas de risco desenhando regiões na imagem da câmera. | Alta |
| RF04 | O sistema deve detectar EPIs em tempo real. | Alta |
| RF05 | O sistema deve identificar ausência de capacete, bota ou mangote. | Alta |
| RF06 | O sistema deve detectar postura ergonômica inadequada. | Alta |
| RF07 | O sistema deve detectar entrada de funcionário em área de risco. | Alta |
| RF08 | O sistema deve gerar alerta visual quando identificar risco. | Alta |
| RF09 | O sistema deve registrar ocorrências de risco. | Média |
| RF10 | O sistema deve gerar relatório de conformidade. | Média |

## Requisitos Não Funcionais

| ID | Requisito | Prioridade |
|---|---|---|
| RNF01 | O sistema deve processar imagens em tempo real. | Alta |
| RNF02 | O sistema deve possuir baixa latência na geração de alertas. | Alta |
| RNF03 | O sistema deve possuir interface simples e operacional. | Média |
| RNF04 | O sistema deve armazenar registros de forma segura. | Alta |
| RNF05 | O sistema deve ser escalável para múltiplas câmeras. | Média |
| RNF06 | O sistema deve funcionar em ambiente industrial controlado. | Alta |
| RNF07 | O sistema deve permitir manutenção e evolução dos modelos de IA. | Média |
