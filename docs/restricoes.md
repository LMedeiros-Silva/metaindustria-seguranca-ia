# Restrições do Sistema

- O sistema depende de câmeras posicionadas corretamente.
- A precisão da IA depende da qualidade do treinamento.
- Ambientes com baixa iluminação podem afetar as detecções.
- O sistema não substitui normas de segurança, apenas apoia a prevenção.
- A primeira versão será testada em ambiente controlado.
- A integração com hardware industrial será planejada para etapas futuras.
